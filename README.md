# Sistema de Monitoramento Inteligente para Cápsula Espacial

## Integrantes

* Eduardo Oliveira-RM:570374
* Joao Pedro-RM:569788

## Descrição

Projeto desenvolvido para a Global Solution utilizando Arduino e conceitos de Internet das Coisas.

## Componentes Utilizados

* Arduino Uno
* Sensor TMP36
* Sensor LDR
* Botão para simulação de vibração
* LEDs
* Display LCD 16x2

## Funcionalidades

* Monitoramento de temperatura
* Monitoramento de luminosidade
* Detecção de vibração
* Alertas visuais por LEDs
* Exibição de dados em LCD

## Tecnologias

* Arduino
* Tinkercad
* GitHub
